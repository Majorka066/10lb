---Название выбранного проекта---
```
Вариант A — Новостная лента (News Feed)
Загрузка новостей через публичный API.
Приложение должно содержать минимальный функционал, состоящий из:
 Лента новостей
 Фильтры по категориям
 Кнопка “обновить”
 Пагинация или “загрузить ещё”
 Модальное окно для просмотра новости полностью
```

---Функционал приложения---

Приложение реализует минимальный, но полноценный функционал новостного сервиса:

---Лента новостей---  
Новости подгружаются с сервера и выводятся в виде карточек.

---Фильтры по категориям---
Пользователь может переключаться между категориями:
- Общее  
- Технологии  
- Бизнес  
- Спорт  
- Наука  
- Здоровье  
- Развлечения  

---Кнопка «Обновить»---
Принудительно перезагружает новости текущей категории.

---Кнопка «Загрузить ещё»---  
Пагинация реализована через подгрузку следующих страниц.

---Модальное окно---  
При клике по карточке открывается модалка с полной информацией:
- Изображение  
- Заголовок  
- Источник  
- Дата  
- Полный текст  
- Кнопка перехода на источник  

---Адаптивный и красивый UI---
Оформлен вручную через CSS:
- тёмная тема  
- карточки с hover-эффектами  
- аккуратные кнопки  
- стилизованная модалка

Использовано **NewsAPI.org** — бесплатное API для получения новостей.

---Возникшие ошибки при интеграции---

Дублирующиеся ключи в NewsCard

Причина:
Некоторые новости не имели URL → ломал key.
Избежаны повторяющиеся key

Использован:
```
key={article.url || index}
```
